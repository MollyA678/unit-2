About this Map
This map is inspired by my <strong>Intro to East Asian Civilizations</strong> coursework this semester.
As I've been learning about the history and development of major cities across East and Southeast Asia,
it has been challenging to keep track of how populations have shifted over time and how historical events
affected urban development. I created this interactive map to help me visualize these changes, allowing 
me to explore population trends from ancient times to the present. Now you can use it too! Use the slider
to view different time periods, and select a city from the dropdown to locate it. Many of these cities stay
at zero population for a long time. Whether this is because they were actually not inhabited yet, or because
some values at worldcitypop.com were missing data is unclear. There are likely cases of both.

Data merged from https://www.kaggle.com/datasets/umxir9/population-by-cities?resource=download
with the geometry location coming from https://www.kaggle.com/datasets/swahajraza/world-cities-dataset
and the historical data coming from https://www.worldcitypop.com/data.asphttps://www.worldcitypop.com/data.asp
